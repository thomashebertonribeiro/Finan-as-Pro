# Finanças Pro 🚀

O **Finanças Pro** é um dashboard financeiro moderno de alta performance, projetado para oferecer controle total sobre suas finanças com o mínimo de esforço manual.

![Premium UI](https://img.shields.io/badge/Design-Glassmorphism-blueviolet)
![Node Version](https://img.shields.io/badge/Node.js-v18+-green)
![SQLite](https://img.shields.io/badge/Database-SQLite-blue)

## ✨ Destaques
- **Automação via WhatsApp**: Registre seus gastos por texto ou enviando fotos de comprovantes no chat.
- **Extração com IA (Google Gemini 2.5)**: Alta precisão na leitura de faturas, identificando bancos e limpando dados indesejados automaticamente.
- **Painel do Agente IA**: Controle total sobre o "System Prompt", logs em tempo real e configurações da IA diretamente na interface.
- **Inteligência de Fornecedores**: Categorização automática de gastos baseada em nomes de lojas/serviços.
- **Visual Premium**: Interface responsiva com design moderno e gráficos interativos.
- **Independência de Nuvem**: Seus dados ficam no seu computador em um banco SQLite.

## 🛠️ Tecnologias
- **Frontend**: React, Vite, Recharts, Lucide-React.
- **Backend**: Node.js, Express, Baileys (WhatsApp Bot), Tesseract.js (OCR).
- **Banco de Dados**: SQLite (Local).

## 🚀 Como Iniciar
Se você já configurou o ambiente seguindo o [SETUP.md](SETUP.md), basta rodar:

```bash
./start_system.bat
```
